Nunux Keeper allows you to create documents in several ways:

- From scratch, starting from a [blank document](blank)
- From a [web page](web-page) by automatically extracting significant content
  and removing unwanted content (ads, trackers, formes, etc.)
- From a part of a web page using the [Bookmarklet](bookmarklet)
- Finally, by taking a [screenshot](web-page) from a web page

{{% notice note %}}
Please note that the images are also saved next to the document.
{{% /notice %}}
